Реализуй всё по техническому заданию ниже:

Проект: LLM-агент для проектирования и редактирования 3D-моделей во FreeCAD
Язык реализации: Python 3.10+
Цель: Создать автономный агент, который:

Принимает от пользователя текстовое ТЗ (на естественном языке) и, опираясь на LLM, генерирует Python-скрипты для FreeCAD.

Выполняет эти скрипты, изменяя/создавая 3D-модель (детали и сборки).

Автоматически делает рендеры (изображения) модели и отправляет их обратно в LLM.

В случае ошибок или некорректной геометрии организует несколько итераций:

анализ текста ошибок и изображения,

запрос дополнительных проекций/зумов,

генерация исправленного скрипта,

повтор исполнения.

Умеет работать со сборками (Assembly3/Assembly4/A2plus), знать версию FreeCAD и установленные расширения, и использовать их для решения задач.

1. Окружение и поддерживаемые платформы

Операционные системы:

Windows 10/11 x64

Ubuntu 22.04 x64

ПО и версии:

FreeCAD не ниже 0.21 (путь до исполняемого файла задаётся в конфиге).

Python 3.10+ (отдельная среда, не обязательно встроенный питон FreeCAD).

LLM с поддержкой:

текстового ввода/вывода,

анализа изображений (мультимодальность, напр. модель типа gpt-4o).

Установка зависимостей:

Через pip (requirements.txt).

Через системные менеджеры пакетов (apt, choco/winget) в установочных скриптах.

2. Общая архитектура

Создать Python-проект, условное название: freecad_llm_agent.
